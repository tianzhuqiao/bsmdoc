7.2 Code block

For code syntax highlighting, bsmdoc uses pygments for parsing. You have to install pygments packages before using this feature.

8 Options

Option Description Default
title the title of the file filename
doctitle the title of the doc
subtitle the subtitle of the doc
orderhead order the heading lines 0
orderheadstart the start level for ordered header 1
utf8conv convert the doc to UTF8 coding 0
utf8convcode code page for UTF8 converting gb2312
eqnsupport support the latex equation 1
eqndir the equation output directory eqs
eqndpi the dpi for the generated equation image 130
eqncache enqble the equation cache 1
eqnpackage the package to be included
eqnfilepre the prefix for the equation file name
config the configure file name
addcss add more css file links
addjs add more javascript file links
